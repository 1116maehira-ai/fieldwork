🎧 センターコール｜アタックリスト作成ツール

テレアポ代行サービス営業用のアタックリストを登録します。FIELDWORK等で集めた企業CSVをアップロードしてSupabaseに保存します

1使い方
📋 想定する流れ：
① FIELDWORK で「リフォーム会社」「塗装会社」など、テレアポが必要そうな業種を Google Maps API で収集
② FIELDWORK の「CSVエクスポート」でCSVをダウンロード
③ そのCSVを下の枠にアップロード
④ 「Supabase に登録」を押すと、centercall_attacks テーブルに保存されます
⑤ 🎧 センターコール｜アタック画面（centercall.html） で電話業務スタート
2CSVをアップロード
📊 読み込み内容
3Supabase に登録
💡 既存企業（電話番号一致）はスキップされます。新規企業のみ追加されます。
📝初回利用時のセットアップ
Supabase で以下のSQLを1回だけ実行してください：

  city text,
  district text,
  email text,
  instagram_url text,
  facebook_url text,
  x_url text,
  tiktok_url text,
  indeed_url text,
  job_antenna_url text,
  agre_url text,
  mynavi_url text,

  ceo_name text,
  founding_date text,
  established_date text,
  revenue text,
  employees text,
  star_rating int DEFAULT 0,
  status text DEFAULT 'untouched',
  next_date date,
  next_action text,
  pending_callback boolean DEFAULT false,

ALTER TABLE centercall_attacks ENABLE ROW LEVEL SECURITY;

CREATE POLICY "cc_select" ON centercall_attacks FOR SELECT USING (true);
CREATE POLICY "cc_insert" ON centercall_attacks FOR INSERT WITH CHECK (true);
CREATE POLICY "cc_update" ON centercall_attacks FOR UPDATE USING (true);
CREATE POLICY "cc_delete" ON centercall_attacks FOR DELETE USING (true);

CREATE INDEX IF NOT EXISTS idx_cc_phone ON centercall_attacks(phone);
CREATE INDEX IF NOT EXISTS idx_cc_status ON centercall_attacks(status);
CREATE INDEX IF NOT EXISTS idx_cc_next_date ON centercall_attacks(next_date);

CREATE TABLE IF NOT EXISTS centercall_scripts (
  id uuid DEFAULT gen_random_uuid() PRIMARY KEY,
  script_id text UNIQUE NOT NULL,
  name text NOT NULL,
  variant text,
  script_json jsonb,
  is_active boolean DEFAULT true,
  sort_order integer DEFAULT 0,
  updated_at timestamptz DEFAULT now()
);
